##### Разметка текста

| **Тип**                      | **Начало** | **Конец** |
|------------------------------| ---------- | --------- |
| Выравнивание по центру       | `::BC` | `::EC` |
| Выравнивание по левому краю  | `::BL` | `::EL` |
| Выравнивание по правому краю | `::BR` | `::ER` |
|------------------------------| ------ | ------ |
| Зачеркнутый текст            | `::BS` | `::ES` |
| Подчеркнутый текст           | `::BU` | `::EU` |
|------------------------------| ------ | ------ |
| Курсивный текст              | `::BI` | `::EI` |
| Жырный текст                 | `::BB` | `::EB` |
